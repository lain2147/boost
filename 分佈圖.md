# 📘 **3D 版本資料處理與圖表製作流程（Markdown 版）**

---

## #️⃣ **Step 1 — 匯入原始資料（Raw Data）**

將量測到的所有 Vdc 數據放到 Excel A 欄：

```
5.243
5.218
5.237
5.231
5.237
…
```

---

## #️⃣ **Step 2 — 小數四捨五入（到小數第 2 位）**

你的規則：

- 若小數剛好 **第 3 位結束** → 不需四捨五入

  - 例如：`5.212` → 可直接 `5.212`（但最後仍統一 round 到 2 位）

- 若超過第 3 位 → 第 3 位四捨五入

  - 例如：`5.2126 → 5.21`

👉 本次資料統一採：

```
=ROUND(A2, 2)
```

---
-數值 ≥ 1 或 ≤ -1：
→ 一律 四捨五入到小數第 2 位

例：5.243 → 5.24

例：5.206 → 5.21

0 到 1 之間（含負數同理）再細分：

若 原本只有到小數第 3 位
→ 完全不動，不四捨五入

例：0.015 → 保持 0.015

例：0.002 → 保持 0.002

若 超過第 3 位（第 4 位以後）才做四捨五入到「第 3 位」
→ 例：

0.0156 → 0.016

0.0025 → 0.003

0.0015 → 0.002

## #️⃣ **Step 3 — 排序（由小到大）**

將四捨五入後的數據排序：

```
5.20
5.206
5.212
5.218
5.225
5.231
…
5.28
```

---

## #️⃣ **Step 4 — 計算每個 Value 的頻率（Frequency）**

建立兩欄：

| Value | Frequency |
| ----- | --------- |

Excel 計算方式：

```excel
=COUNTIF($A:$A, D2)
```

得到如下表：

| Value | Frequency |
| ----- | --------- |
| 5.20  | 1         |
| 5.21  | 8         |
| 5.22  | 25        |
| 5.23  | 15        |
| 5.24  | 66        |
| 5.25  | 19        |
| 5.26  | 12        |
| 5.27  | 2         |
| 5.28  | 2         |

---

## #️⃣ **Step 5 — 建立 Spec 上限 / 下限 欄位**

建立：依照每個單元參數對照不一定有預設沒有

- **SpecMinBar（綠色）**
- **SpecMaxBar（紅色）**

### ✔ 規則：

- 若 Value = 下限（5.2）→ 填入最大頻率（滿版虛線）
- 若 Value = 上限（5.5）→ 若超出資料範圍 → 留空即可不顯示
- 其他皆留空

| Value | Freq | SpecMinBar | SpecMaxBar |
| ----- | ---- | ---------- | ---------- |
| 5.20  | 1    | 66         |            |
| 5.21  | 8    |            |            |
| 5.22  | 25   |            |            |
| 5.23  | 15   |            |            |
| 5.24  | 66   |            |            |
| 5.25  | 19   |            |            |
| 5.26  | 12   |            |            |
| 5.27  | 2    |            |            |
| 5.28  | 2    |            |            |

---

## #️⃣ **Step 6 — 插入 3D 直條圖（Column 3D）**

1. 插入 → **3D 柱狀圖**
2. 第一組：藍色 → Frequency
3. 第二組：綠色虛線柱 → SpecMinBar
4. 第三組：紅色虛線柱 → SpecMaxBar
5. 再加入折線圖系列（Freq Line）

---

## #️⃣ **Step 7 — 設定顏色與樣式**

### **Frequency（柱）**

- 藍色
- 3D 樣式可自由調整

### **Freq Line（折線）**

- 紫色
- 平滑或折線皆可

### **Spec Min（下限）**

- 線條色：**綠色**
- 虛線
- 填滿：無（透明）
- 高度 = 最大頻率（滿版）

### **Spec Max（上限）**

- 線條色：**紅色**
- 虛線
- 填滿：無
- 若不在資料範圍 → 自動不顯示

---

## #️⃣ **Step 8 — 完成最終 3D 量測圖**

你將得到：

- 3D 柱狀圖（藍）
- 量測折線（紫）
- 下限綠色虛線
- 上限紅色虛線（在範圍內才出現）

完整符合你的 Vdc 曲線分析與報表格式。
